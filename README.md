# Thirtysixstudio Project

Welcome to the Thirtysixstudio project! This application is built using React and Vite, providing a seamless and efficient development experience with Hot Module Replacement (HMR) and optimized build processes.

## Project Overview

Thirtysixstudio is a boutique production studio focused on creating digital assets and immersive experiences for purposeful brands. Our application showcases our expertise in design, animation, and technology, constantly rethinking what digital craft can do for modern-day ads and campaigns.

## Key Features

- **Dynamic Canvas Animations**: Utilizes GSAP for smooth and engaging animations.
- **Responsive Design**: Ensures a seamless experience across all devices.
- **Interactive User Interface**: Engages users with interactive elements and transitions.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **Vite**: A build tool that provides a fast and lean development environment.
- **GSAP**: A powerful animation library for creating high-performance animations.
- **Locomotive Scroll**: For smooth scrolling effects and animations.
